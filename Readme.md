![Screenshot of Fluster](/assets/images/fluster-screenshot.png)

Fluster is a native Desktop client for Fluxer, a Discord alternative with a lot of potential.

It uses my own immediate-mode GUI framework called [uing](https://git.speykious.dev/Speykious/uing).

The goal is to have a client that is faster, more lightweight and more efficient than Fluxer's main
desktop app, while also looking 99.9% like Fluxer.
